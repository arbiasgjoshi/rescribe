# Rescribe

Rescribe is a site generator for Remix, that lets you build documentation and blog sites with ease. It leverages Remix's features and provides the components and utilites to make building content-focused sites faster.

You can install Rescribe in any Remix site and have full-featured docs and blog in minutes.
