Rescribe is designed to be used with minimal effort. You just have to write some configuration and copy-paste some code — which you can find here — to get it up and running.

Let's start with installing the package:

```bash
pnpm add @rescribejs/core @rescribejs/server
```

### Next Steps

Now you can start [configuring your CMS](/docs/config).
