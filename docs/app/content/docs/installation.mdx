Rescribe is designed to be used with minimal effort. There are some features that will require you to copy-paste some code that will be shown here in the docs.

### Installation

Let's start with installing the package:

```bash
pnpm add @rescribejs/core @rescribejs/server
```

### Next Steps

Now you can start configuring your docs/blog site or you check out the configuration that is site uses.
